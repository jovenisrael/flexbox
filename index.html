Introduction to Flexbox

CSS Flexbox (Flexible Box Layout) is a layout method that provides an efficient way to arrange, distribute, and align items in a container, even when their size is unknown or dynamic.

Key Concepts:
  • Flex Container: The parent element with display: flex
  • Flex Items: The children of the flex container
  • Main Axis: The primary axis along which flex items are laid out
  • Cross Axis: The axis perpendicular to the main axis

Quick Reference

Container Properties

Property Values Description
display flex | inline-flex Creates a flex container
flex-direction row | row-reverse | column | column-reverse Sets the main axis direction
flex-wrap nowrap | wrap | wrap-reverse Controls if items wrap to new lines
justify-content flex-start | flex-end | center | space-between | space-around | space-evenly Aligns items along main axis
align-items stretch | flex-start | flex-end | center | baseline Aligns items along cross axis
align-content stretch | flex-start | flex-end | center | space-between | space-around Aligns wrapped lines
gap length | percentage Sets gaps between items

Item Properties

Property Values Description
flex-grow number (default: 0) How much item should grow
flex-shrink number (default: 1) How much item should shrink
flex-basis auto | length | percentage Initial size before growing/shrinking
flex grow shrink basis Shorthand for grow, shrink, basis
align-self auto | stretch | flex-start | flex-end | center | baseline Override container's align-items
order integer (default: 0) Changes visual order of items

Common Flex Shorthand Values

Shorthand Equivalent to Use Case
flex: 1 flex: 1 1 0% Items grow equally, ignore content size
flex: auto flex: 1 1 auto Items grow/shrink based on content
flex: none flex: 0 0 auto Items don't grow or shrink
flex: 0 0 200px flex: 0 0 200px Fixed size item (200px)
